# reduxform-fields

## Install

`npm install --save reduxform-fields`

## Usage

### Example
```javascript
  import { InputField } from 'reduxform-fields'

  <Field
    component={InputField}
    label="First Name"
    name="firstName"
    type="text"
  />
```
